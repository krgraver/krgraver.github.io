Project Riley

Client: Curious Technologies

Role: UX

Curious Technologies is currently looking into the best way of pairing a remote sensor with a mobile application. The sensor they have in mind is able to collect data using an accelerometer and gyroscope and send it to a nearby device via Bluetooth. The application could then visualize the data in any number of ways. Curious Technologies thought a great use of this pairing would be in the classroom, specifically for high school physics students. They asked my UX team to conduct domain research, create wireframes, and test prototypes for the ideal physics learning application using this technology.
